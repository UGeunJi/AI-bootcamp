- 부스팅으로 바꾸기

  - 나는 바퀴
  - 나는 운전사

8. 물결 표시 \~는 흔히 `17~18일`과 같이 범위를 나타낼 때 많이 쓰는데요. 마크 다운 문법에서는 중간선을 나타낼 때 사용합니다.
- 범위 : 17\~18
- ~기울임~

이럴 때에는, 물결 표시 앞에 \(원표시, 엔터 위에 있는)를 적어주시면 마크다운 문법으로 사용하지 않겠다라는 뜻이 됩니다!
